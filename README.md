# Idioma
Lokaverkefni í Gervigreind THED
Verkefnið snýst um að prófa frammistöðu þriggja risamállíkana í skilningi og notkun orðatiltækja á íslensku.
Gögn sem eru aðgengileg hér:
*hrágögn með niðurstöðum prófana á CSV formi
*niðurstöður netkannana
*gögn úr prófunum omv.
*viðaukar úr lokaverkefni, listi yfir orðtök, kvaðningar, verkefni og spurningar í netkönnunum
